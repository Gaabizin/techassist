⬅ Voltar para Serviços

Como um backup pode te proteger de ataques virtuais.

Podemos realizar backups de algumas formas distintas, mas o mais importante deste método de salvar seus arquivos é que isto seja feito periodicamente e sem falhas.

1. Backup fisico:

O backup fisico é mais demorado pois precisa ser feito manualmente, fazendo uma cópia dos arquivos mais importantes do computador ou de todos arquivos. Porém o backup fisico continua sendo o mais seguro, por ser mais dificil dos arquivos corromperem e também devido a evitar erros de terceiros como em backups na nuvem.

2. Backup digital(Em Nuvem):

Como seu nome mesmo diz é um backup virtual, situado em servidores terceiros de serviços que em sua maioria são pagos, disponibilizando apenas uma certa quantia e qualidade de backup gratuito, são mais práticos e mais rápidos mas podem deixar a desejar devido algumas definições de qualidade.

  • • Backup Completo (Full):
    • o Como funciona: Copia todos os dados selecionados, independentemente de terem sido modificados ou não
    • o Vantagens: Restauração é rápida e simples, exigindo apenas o último backup completo.
    • o Desvantagens: É o mais demorado para realizar e consome mais espaço de armazenamento.
  • • Backup Incremental:
    • o Como funciona: Copia apenas os dados modificados ou criados desde o último backup (seja ele completo ou outro incremental).
    • o Vantagens: Muito rápido de realizar e ocupa menos espaço de armazenamento.
    • o Desvantagens: A restauração é a mais lenta e complexa, pois requer o último backup completo + todos os incrementais subsequentes.
  • • Backup Diferencial:
    • o Como funciona: Copia apenas os dados modificados desde o último backup completo.
    • o Vantagens: Mais rápido que o completo e restauração mais simples que o incremental (apenas completo + último diferencial).
    • o Desvantagens: Ocupa mais espaço que o incremental, pois acumula as alterações
